Kubernetes Commands

Basic Commands:

  • kubectl get pods - List running pods in the cluster.
  • kubectl describe pod <pod_name> - Show details of a specific pod.
  • kubectl apply -f <file>.yaml - Apply a Kubernetes configuration file.
  • kubectl delete pod <pod_name> - Delete a pod.

Advanced Commands:

  • kubectl rollout restart deployment <deployment_name> - Restart a deployment.
  • kubectl get nodes - List nodes in the cluster.
  • kubectl exec -it <pod_name> -- /bin/bash - Run commands inside a pod.
