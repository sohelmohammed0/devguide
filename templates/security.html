Security Testing with Trivy

Trivy Commands

Basic Commands:

  • trivy image <image_name> - Scan a Docker image for vulnerabilities.
  • trivy fs <directory> - Scan the filesystem for vulnerabilities.

Advanced Commands:

  • trivy --severity HIGH,CRITICAL image <image_name> - Scan with severity filters.
  • trivy --ignore-unfixed image <image_name> - Ignore unfixed vulnerabilities.
