Docker & Containerization

Docker Commands

Basic Commands:

  • docker run <image> - Run a container from an image.
  • docker ps - List running containers.
  • docker images - List available Docker images.
  • docker build -t <tag> . - Build a Docker image from a Dockerfile.
  • docker exec -it <container> /bin/bash - Access a running container.

Advanced Commands:

  • docker-compose up - Start services defined in a Docker Compose file.
  • docker swarm init - Initialize Docker Swarm for orchestration.
  • docker network ls - List Docker networks.
  • docker volume ls - List Docker volumes.
